CheatSheet - GIT

Repository in Github erstellen ---> Die 3 Zeilen nicht vergessen (WICHTIG !!!!)

Repos erstellen

git init:

Neues, lokales Repo erstellen

Lokale Änderungen

git add:

Alle lokalen Änderungen zum nächsten Commit hinzufügen

git commit -m:

Zur Staging Area hinzugefügte Änderungen committen inklusive Infos zur vorgenommenen Veränderungen

git status:

Änderungen anzeigen

Branches

git checkout branchname:

Aktuellen HEAD-Branch wechseln

git checkout -b branchname:

Erstellt einen neuen Branch und wechselt direkt zu diesem

git checkout master:

Den Branch wechseln (hier zum Branch master)

Update und Publish
