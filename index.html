CheatSheet - GIT

Repos erstellen

git init:

Neues, lokales Repo erstellen

Lokale Änderungen

git status:

Den aktuellen Stand zu Änderungen, Branches, Commits anzeigen

git add:

Das aktuelle Projekt dem Staging-Bereich hinzufügen

git commit -m:

Zur Staging Area hinzugefügte Änderungen committen inklusive Infos zur vorgenommenen Veränderungen

git fetch origin:

Lädt Änderungen vom Repository origin herunter

git pull:

Den aktuellen Stand des Git-Repositories auf das lokale Repository ziehen/aktualisieren

Branches

git checkout branchname:

Aktuellen HEAD-Branch wechseln

git checkout -b branchname:

Erstellt einen neuen Branch und wechselt direkt zu diesem

git checkout branchname:

Den Branch wechseln

Update und Publish

Merge

git merge branchname:

Der Branch wird mit dem aktuellen Branch zusammengeführt.
VORSICHT !!! Merge kann Konflikte verursachen
